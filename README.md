Employee Management Portal

A Back-End Employee Management System with Simple UI(Thymeleaf) developed during my internship using Spring Boot.

-- Tech Stack --
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Thymeleaf
- HTML / CSS
- Maven

-- Features --
- Employee CRUD operations
- DTO-based request/response handling
- Validation using Jakarta Validation
- Global Exception Handling
- Pagination support
- Simple UI using Thymeleaf
- REST APIs + UI integration

-- Modules --
- Employee Management
- Backend REST APIs
- UI Layer (Thymeleaf)

-- Learning Outcome --
- Clean layered architecture
- DTO pattern
- Pagination & validation
- Real-world error handling
